# Dart Shopping List Manager

This is a simple shopping list manager built using the Dart programming language.  
The program demonstrates basic programming concepts such as lists, loops, and conditional statements.

## Features

- Display all shopping items
- Add new items to the list
- Remove items from the list
- Search for an item in the list
- Sort items alphabetically
- Count the total number of items

## Technologies Used

- Dart Programming Language

## Program Functionality

The program starts with a predefined list of shopping items.  
It then performs several operations such as adding a new item, removing an existing item, searching for an item in the list, and sorting the items alphabetically.

## Concepts Demonstrated

- List data structure
- For loops
- Conditional statements
- List operations (add, remove, contains, sort)
